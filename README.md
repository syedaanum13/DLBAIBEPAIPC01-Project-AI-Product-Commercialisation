### DLBAIBEPAIPC01-Project-AI-Product-Commercialisation
### Overview
This project set out to develop a data-driven forecasting prototype for weekly avocado prices in the 
United States, with the dual objective of demonstrating technical proficiency in time-series modelling 
and translating the resulting outputs into meaningful business insights.

### Folder Structure
1. `AvoCast-Business-Canvas-Model` - This folder contains the business canvas model
2. `AvoCast-Jupyter-Notebook` - This folder conttains the Jupyter Notebook
3. `AvoCast-Project-Planning` - This folder contains the Trello project planning board
4. `AvocCast-Visualization` - This folder contains all the graphs

### Getting Started
Follow these steps to run the project locally.

#### 1. Clone the Repository
   
Execute the command to clone the Git Repo: `git clone https://github.com/syedaanum13/DLBAIBEPAIPC01-Project-AI-Product-Commercialisation.git`

Execute the command to change the directory: `cd DLBAIBEPAIPC01-Project-AI-Product-Commercialisation/AvoCast-Jupyter-Notebook`

#### 2️. Install Required Libraries

Make sure you have Python installed, then run:

`pip install notebook`

#### 3️. Run Jupyter Notebook

Start the notebook server:

`jupyter notebook`

Then open the file:

`Avacado.ipynb`

#### 4️. Run the Notebook

Press `Shift + Enter` to execute cells step by step
